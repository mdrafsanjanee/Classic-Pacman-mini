# 🟡 Classic PacMan Game in C

A simple console-based **PacMan game** built using **pure C programming**, focusing on **basic logic, arrays, and fundamental concepts**. This project is designed as a **mini project** to demonstrate understanding of loops, conditions, and array manipulation.

---

## 🎯 Project Objective

The goal of this project is to:

* Practice core C programming concepts
* Understand game logic using **2D arrays**
* Implement movement and collision detection
* Build a fun, interactive console application

---

## 🧠 Concepts Used

* Arrays (1D & 2D)
* Loops
* Conditional statements
* Functions
* Basic input/output
* Game logic handling

---

## 🎮 Features

* 🟡 Player-controlled PacMan movement
* 👻 Enemy movement 
* 🧱 Maze represented using 2D arrays
* 🍒 Food/points collection system
* 💯 Score tracking
* ❌ Game over condition (collision with enemy)

---

## 🕹️ Controls

| Key | Action     |
| --- | ---------- |
| W   | Move Up    |
| S   | Move Down  |
| A   | Move Left  |
| D   | Move Right |
